# skills

Personal collection of LLM skills for Claude Code / Codex. Each skill lives in its
own top-level directory containing a `SKILL.md`. Skills are linked into the local
skills path (`~/.claude/skills` or `~/.codex/skills`) via the `sync-my-skills` skill.
